# FIRE Calculator

A feature-rich **Financial Independence, Retire Early** calculator with tax-aware projections, Monte Carlo simulation, and multi-scenario comparison.

**[Live Demo](https://fire-calculator-f0y.pages.dev)**

## Features

- **Tax-Aware Calculation** — Built-in progressive tax brackets for China, US, Hong Kong, and Singapore. Input pre-tax income, see after-tax breakdown instantly.
- **Salary Growth** — Annual salary growth rate compounds year-over-year for realistic projections.
- **Capital Gains Tax** — Configurable investment returns tax rate.
- **5 FIRE Types** — Lean / Regular / Fat / Coast / Barista FIRE side-by-side comparison.
- **Monte Carlo Simulation** — 100-10,000 random market simulations with success probability and P10-P90 wealth distribution fan chart.
- **Life Events Timeline** — Add one-time income/expenses, salary changes, passive income at specific ages.
- **Scenario Comparison** — Up to 4 independent scenarios with overlay charts and comparison table.
- **Multi-Currency** — CNY, USD, EUR, GBP, JPY, HKD, SGD, AUD, CAD with localized formatting.
- **Bilingual** — Chinese / English, one-click switch.
- **Dark Mode** — System-aware with manual toggle.
- **Persistent** — All inputs auto-saved to localStorage.

## Tech Stack
